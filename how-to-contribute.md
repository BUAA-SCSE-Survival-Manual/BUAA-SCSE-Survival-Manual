# github + MR


## STEP.1 install node.js

为了在本地编辑的时候，能够看到编辑后的效果，需要安装node.js

推荐linux环境下维护

## STEP.2 install gitbook-cli

```shell
npm install -g gitbook-cli
```


## STEP.3 clone repo

```shell
git clone xxx
```

## STEP.4 edit & render

编辑后，查看效果
```shell
gitbook serve
```

实时编辑，实时渲染

## STEP.5 commit & create MR

提交MR
